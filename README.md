# Letícia Toledo Melo

Desenvolvedora .NET focada em construir aplicações web com ASP.NET Core, Entity Framework Core e boas práticas de engenharia. Apaixonada por código limpo, automação e UX simples e objetiva.

## Stack Principal
- .NET (ASP.NET Core, Minimal APIs, MVC, Razor)
- Entity Framework Core (SQL Server, InMemory para testes)
- AutoMapper, FluentValidation, xUnit/NUnit
- Azure (App Service, SQL Database) e CI/CD com GitHub Actions

## Destaques Rápidos
- Projetos com camadas: Domain, Infrastructure, Web, Services
- Mapeamentos com EF Core e perfis de AutoMapper
- Pipelines de CI para build e validação

## Projetos em Destaque
- Cadastro (ASP.NET Core + EF Core) — CRUD simples e organizado
- APIs REST com documentação e versionamento
- Templates de projetos prontos para escalar em .NET
